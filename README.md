# 🏠 House Price Predictor

AI-powered house price prediction using Machine Learning.

## Features
- Machine Learning (Linear Regression)
- FastAPI REST API
- RAG Document Q&A System

## Tech Stack
- Python
- scikit-learn
- FastAPI
- LangChain
- Ollama

## How to Run
```
pip install -r requirements.txt
uvicorn main:app --reload
```
